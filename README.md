# homebrew-tap

Homebrew tap for projects by [@alyraffauf](https://github.com/alyraffauf).

## Usage

```
brew tap alyraffauf/tap
```

## Formulae

| Formula | Description |
|---------|-------------|
| `atbbs` | AT Protocol bulletin board system |
| `dewy`  | Minimal terminal UI for Todoist |

### atbbs

```
brew install atbbs
```

### dewy

```
brew install dewy
```
